ObserveRTC Demo Application
---

## Clone

```bash
git clone https://github.com/ObserveRTC/demo-app.git
```

## Install and Build

1. Install media-server

```bash
cd media-server && yarn
```

2. Install the webapp

```bash
cd webapp && yarn
```

## Run

1. Start sfu

```bash
cd sfu

node dist/main.js
```

2. Start the webapp

```bash
cd webapp

yarn start
```
